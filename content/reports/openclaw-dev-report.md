## 四、技术特点

### ✅ 优势
1. **自架部署**：数据完全掌控，无需依赖第三方
2. **多通道统一**：单一 Gateway 管理多个消息平台
3. **Agent 原生**：专为 AI Agent 设计的架构
4. **开源 MIT**：社群驱动，持续迭代
5. **跨平台**：macOS/Linux/Server 皆可运行

### ⚠️ 挑战
1. 需要技术门槛（Node.js 22+）
2. 需要自备 API Key
3. 部署维护需自行处理
